# RCS
LMS7기, C# WPF를 활용한 차량 원격 제어 시스템

## 1팀 GitHub 규칙

- main에 작업 금지, main은 병합용입니다.
- 각 파트별 브랜치 생성(혹은 팀원별) 후 작업해주세요.
- commit 메시지 작성 방법에 따라 commit 후 push 하세요.
- pull requests 할 때에는 팀원들에게 알리고 진행해주세요.

## Commit 메시지 작성 방법

"{commit 날짜}, {코드작성자}, {Commit 메시지}"

## 프로토콜(JSON)

- 헤더: 항상 4바이트 (바디 크기)
- 바디: JSON 문자열 (UTF-8)
- 바디 최대 크기: 100KB
